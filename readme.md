# Superstore Sales Analysis

Este proyecto analiza datos de ventas de una tienda Superstore con el objetivo de identificar patrones de ventas, ganancias, descuentos y rentabilidad.

## Objetivo

Analizar qué categorías, subcategorías, regiones, segmentos de clientes y niveles de descuento están asociados a mejores o peores resultados comerciales.

## Herramientas utilizadas

- Python
- Pandas
- Plotly
- Jupyter Notebook
- Git y GitHub

## Contenido del repositorio

- `data/`: contiene el dataset utilizado.
- `notebooks/`: contiene el notebook principal del análisis.
- `images/`: contiene evidencias del uso de Git y GitHub.
- `requirements.txt`: contiene las librerías necesarias para ejecutar el proyecto.

## Principales análisis realizados

- Limpieza básica de datos.
- Conversión de fechas.
- Creación de variables derivadas.
- Análisis temporal de ventas y ganancias.
- Análisis de categorías y subcategorías.
- Análisis de descuentos y rentabilidad.
- Mini dashboard con resultados principales.
